# Sistema de Venta de Ropa al por Mayor

Aplicativo web monolítico MVC en PHP 8.3+ y MySQL 8.

## Estructura
El proyecto sigue una arquitectura MVC estricta. Todo el tráfico web debe ser dirigido a la carpeta `/public`.

## Instalación en XAMPP
1. Clonar este repositorio en `htdocs`.
2. Configurar el VirtualHost o acceder a `localhost/ruta_al_proyecto/public/`.
3. Importar el script SQL ubicado en `/database/schema.sql`.
4. Importar los datos iniciales con `/database/seed.sql`.

*Ver los documentos de diseño en la carpeta de especificaciones para más detalles.*
